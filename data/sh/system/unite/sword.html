合体剣とは？

合体剣とは？

レオン自工クリア後にマダム銀子から譲り受ける「無銘の刀」を悪魔と合体させることで強力な武器へと変化させることができる。

合体させる悪魔のLVと力によって攻撃力が変化するので、強い悪魔を合体させればそれだけ強力な剣が出来上がる。勿論御霊強化も反映される。

なお、無銘の刀や合体剣を手放した場合はビデオマッスルで購入することができる。

剣の能力

剣の能力

攻撃力にはLVと力、命中力にはLVと速さと運が関係する。

  • 攻撃力 = LV + ( 力 * 2 )
  • 命中力 = ( LV * 0.8 ) + ( 速さ * 0.8 ) + ( 運 * 0.4 )

但し、剣の種類によって攻撃力・命中力共に以下の割合で修正を受ける。

  • 氷刃・電光 = 0.95
  • 毒刃・夢想・魔映 = 0.9

* 計算は全て切り捨て

特殊合体
